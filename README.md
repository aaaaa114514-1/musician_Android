# Musician for Android

**Musician: A Simple Music Player**

---

## How to Download and Install

- Download the installation file `app-debug.apk`from the repository path `app\build\outputs\apk\debug`
- The app can be installed directly on devices running **Android** or **HarmonyOS** versions **before** `NEXT`(excluding `NEXT`)

---

## Supported Features

- Reads the local directory for `.mp3` files and adds them to the playlist
- Play / Pause / Previous Track / Next Track
- Shuffle Mode / Single Loop / List Loop
- Dark Mode / Light Mode
